# Lecture Outline

1. The server is the student's personal internet home.
2. Cloudflare owns the public edge; Oracle VM owns the app runtime.
3. Next.js handles user experience, SEO pages, metadata, sitemap, and public routing.
4. Flask owns durable business capabilities: auth, users, blog records, email, AI jobs.
5. Google Sign-In proves identity; backend creates the user account.
6. Newsletter is a permission system: collect email, confirm intent, send useful updates.
7. Daily AI blogs are not random text; they are a content pipeline with prompts, review modes, metadata, and publishing rules.
8. OpenClaw/Codex can maintain the system because the API contract, env vars, jobs, and deployment units are explicit.
